box.py

dlib 라이브러리를 사용하여 얼굴을 검출하고, 각 얼굴의 얼굴 랜드마크를 식별하여 나타냄

sticker.py

얼굴의 랜드마크를 기반으로 이미지에 스티커를 붙임

main.py

얼굴 랜드마크를 기반으로 스티커를 얼굴에 붙임. (미완성)
